1. Download https://openjfx.io/

2. Install e(fx)clipse from Marketplace

3. JavaFX as User Library:

Preferences -> Java -> Build Path -> User Libraries -> New
Project -> Properties -> Java Build Path -> Add Library -> User Library

4. Run Configuration:

Execution environment:
	Java-SE 11
VM arguments:
	-Defxclipse.java-modules.dir="C:\Program Files\AdoptOpenJDK\javafx-sdk-11.0.2\lib"
